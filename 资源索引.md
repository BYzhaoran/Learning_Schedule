# 资源索引（按关键部分 + 学习顺序）

来源说明：
- 优先参考 Embodied-AI-Guide: <https://github.com/BYzhaoran/Embodied-AI-Guide#info>
- 每个模块都给出“先学/后学”顺序，避免资源堆叠

使用规则：
- 每月每个主模块只选2-3个精学资源
- 每个资源必须绑定产出（代码/实验/笔记）
- 当月无实验结果时，下月减少理论刷课，增加实现与调参
