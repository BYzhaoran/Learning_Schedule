# M12 SAC/TD3 + MuJoCo任务封装

## 本月定位
- 主题：离策略算法、连续控制、环境封装
- 前置依赖：把 DRL 从离散任务推进到连续控制。

## 核心任务
1. 实现 SAC 或 TD3 并完成训练
2. 封装 MuJoCo 任务的训练与回放
3. 导出模型与评估脚本

## 关键实验
- SAC 与 TD3 在同任务上做对比。

## 交付物
- MuJoCo 训练与回放工具链。

## 验收标准
- 可训练、可回放、可导出。
