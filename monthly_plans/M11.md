# M11 PPO实战

## 本月定位
- 主题：策略梯度、优势估计、剪切目标
- 前置依赖：在 DQN 基础上进入策略优化。
- 本月目标：掌握 on-policy 方法的稳定训练技巧与超参直觉。

## 必学知识点

### 1. 策略梯度核心
1. REINFORCE 到 Actor-Critic 的演进。
2. Advantage 的方差降低作用。
3. on-policy 与 off-policy 的样本效率差异。

### 2. PPO 机制细节
1. clip objective 的约束意义。
2. value loss、policy loss、entropy loss 的平衡。
3. GAE($\lambda$) 对偏差与方差的影响。

### 3. 实践稳定性
1. rollout 长度与 minibatch 更新轮数耦合。
2. 优势归一化与奖励缩放策略。
3. 连续动作空间下的方差参数控制。

## 主线任务（必做）
1. 实现 PPO 训练循环与日志系统。
2. 在至少 1 个连续控制任务上稳定收敛。
3. 完成 clip 与 GAE 超参数分析。
4. 增加 entropy 系数与 value loss 系数调节实验。

## 扩展任务（建议做）
1. 增加并行环境采样提高样本吞吐。
2. 对比 PPO 与 DQN 在离散任务上的行为差异。
3. 实现 early stopping（基于 KL）避免策略崩溃。

## 关键实验
1. 至少 4 组 PPO 超参数消融。
2. clip ratio 不同设置对收敛稳定性影响。
3. GAE 参数（$\lambda$）对学习速度与方差影响。
4. entropy 系数变化对探索行为影响。

## 交付物
1. PPO 实现代码与训练配置。
2. 实验报告（收敛曲线 + 超参分析）。
3. 一份 PPO 调参速查表。

## 完成标志
1. 至少 1 个任务稳定收敛。
2. 能解释 PPO 崩溃常见原因（过更新、估计偏差、探索不足）。
3. 能给出一组可迁移到新任务的默认超参。
