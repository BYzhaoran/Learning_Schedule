# monthly_plans索引

这个目录放 24 个月的执行卡片。原则是：
- README 只负责导航和统一规则
- 每个月文件只保留和当月相关的任务，不再重复周模板、风险模板和复盘模板
- 详细阶段关系见 [roadmap.md](../roadmap.md)，资源入口见 [resource_index.md](../resource_index.md)

## 使用方式
1. 先看 [roadmap.md](../roadmap.md) 确认当前所处阶段。
2. 打开对应月份文件，直接执行当月任务。
3. 只保留月度差异，不把通用模板重复写进每个文件。
4. 月末用同一套复盘问题：做成了什么、卡在哪里、下月改什么。
