# M18 多智能体强化学习基础

## 本月定位
- 主题：多智能体决策、价值分解、通信与协作
- 前置依赖：掌握单智能体 DRL（M13-M17）、分布式训练（M14）。
- 本月目标：从单智能体扩展到多智能体，理解竞争与协作的动力学。

## 技术主线
1. 从单智能体 Markov 决策到多智能体 Markov 博弈。
2. 掌握集中式与分散式策略执行框架（CTDE）。
3. 学习价值函数分解与通信机制。

## 必学技术点

### 1. 多智能体 RL 问题
1. 非平稳环境与其他智能体的影响。
2. 奖励结构：完全合作、完全竞争、混合设定。
3. 部分可观测与通信的必要性。

### 2. 核心算法
1. QMIX：价值函数单调分解（monotonic value decomposition）。
2. MADDPG：多智能体 Actor-Critic 与中央化评论家。
3. 独立学习者与显式通信的权衡。

### 3. 多智能体学习动力学
1. 收敛性：纯策略、混合策略与平衡条件。
2. 通信拓扑与信息流的影响。
3. 智能体间的行为影响与非稳定性。

## 核心任务
1. 实现独立 DQN 或 QMIX 来解决多智能体问题。
2. 选定合作环境（如 SMAC、自建网格世界），训练 3-5 个智能体。
3. 对比独立学习、集中式训练、有无通信的性能。
4. 分析协作行为的涌现与失败模式。

## 关键实验
1. 独立学习 vs 集中式训练 vs QMIX 的收敛性对比。
2. 智能体数量扩展（2 个 → 5 个）的可扩展性实验。
3. 通信拓扑（全连接 → 局部 → 无）对策略协调的影响。
4. 奖励结构变化（完全合作 → 混合）对收敛的影响。

## 交付物
1. 多智能体学习代理实现（QMIX 或 MADDPG）。
2. 多智能体环境与可视化工具。
3. 协作与竞争动力学的分析报告。

## 验收标准
1. 多智能体策略稳定收敛（至少 3 个智能体）。
2. 能解释 QMIX 中的价值分解机制与约束。
3. 观察并分析行为的涌现与竞争/协作权衡。
