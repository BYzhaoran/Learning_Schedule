# M13 TRPO 与信任域策略优化

## 本月定位
- 主题：信任域算法、KL 散度约束、策略优化理论深入
- 前置依赖：掌握 PPO 和策略梯度基础（M10-M12）。
- 本月目标：理解策略优化的约束边界，为后续算法打好理论基础。

## 技术主线
1. 从无约束梯度上升理解到有约束优化框架。
2. 掌握 KL 散度约束、自然梯度、Fisher 信息矩阵的几何意义。
3. 实现 TRPO 并对比与 PPO 的收敛行为与鲁棒性。

## 必学知识点

### 1. 信任域优化理论
1. 策略性能差分定理与 advantage 表示。
2. KL 散度约束的作用：防止策略剧变。
3. 自然梯度与 Fisher 信息矩阵的计算与近似。
4. 单调改进保证的推导与限制条件。

### 2. TRPO 核心机制
1. 约束优化问题的二阶近似与海森矩阵。
2. 共轭梯度法（CG）求解二次优化。
3. 回溯线搜索（backtracking line search）保证单调性。
4. 超参数：δ（KL 上界）、回溯系数、CG 迭代次数。

### 3. 实现与调试
1. Fisher-Vector 乘积的高效计算（不显式构造矩阵）。
2. Hessian 或 Fisher 逆矩阵的稳定化（damping factor）。
3. 梯度检查与数值验证。

## 核心任务
1. 实现 TRPO 训练循环，包括 advantage 估计、Fisher 信息计算、CG 求解、回溯搜索。
2. 在 CartPole、LunarLander、连续控制任务各跑 1 次，对比收敛速度与稳定性。
3. 做 TRPO vs PPO 的关键对比实验（KL 约束的影响、样本效率、鲁棒性）。
4. 形成调参手册（δ、CG 精度、damping 系数对收敛的影响）。

## 关键实验
1. TRPO vs PPO 在同一任务上的收敛曲线对比（至少 3 个任务）。
2. KL 上界 δ 的敏感性分析（δ = 0.01, 0.001, 0.0001）。
3. CG 精度（max_iters）与最终性能的权衡实验。
4. 不同初始化下的稳定性对比。

## 交付物
1. TRPO 实现代码（含详细注释）。
2. TRPO vs PPO 对比报告（图表 + 性能指标）。
3. 调参速查表与实现细节说明。

## 验收标准
1. TRPO 实现可复现，至少 1 个任务稳定收敛。
2. 能解释 TRPO 与 PPO 的理论与实践差异。
3. 理解信任域约束如何保证单调改进。
