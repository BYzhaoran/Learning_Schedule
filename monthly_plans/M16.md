# M16 离线强化学习与批量数据优化

## 本月定位
- 主题：离线 RL、分布移位问题、批量约束算法
- 前置依赖：掌握在线 RL 与 DRL 基础。
- 本月目标：学习从固定数据集进行策略学习，处理分布不匹配问题。

## 技术主线
1. 理解离线 RL 的核心挑战：外推误差与分布移位。
2. 掌握约束优化方法（CQL、IQL、Conservative Updates）。
3. 在标准离线 RL 基准上验证算法性能。

## 必学技术点

### 1. 离线 RL 问题设置
1. 分布外（OOD）行为与外推误差的形成。
2. 批量约束的必要性与形式。
3. 行为策略的未知与动态假设。

### 2. 核心算法
1. Conservative Q-Learning (CQL)：Q 函数下界估计。
2. Implicit Q-Learning (IQL)：值函数与策略解耦。
3. 其他方法：AWR、CRR、Diffusion Policy。

### 3. 评估与对比
1. 离线评估的困难与 doubly robust 估计。
2. 在线微调的必要性与成本。
3. Offline-to-Online RL 的渐进式学习。

## 核心任务
1. 实现 CQL 或 IQL（至少选一个完整实现）。
2. 在 D4RL 数据集上完成 offline RL 实验。
3. 与行为策略与上限（最优策略）做性能对比。
4. 分析数据质量与覆盖率对性能的影响。

## 关键实验
1. 不同数据质量下的离线 RL 性能对比。
2. CQL/IQL 中约束强度对收敛与最终性能的影响。
3. Offline-to-Online 微调的收敛速度。
4. 不同环境（locomotion vs manipulation）的性能差异。

## 交付物
1. CQL/IQL 实现代码。
2. D4RL 数据集实验结果与对比。
3. 离线 RL 性能分析报告。

## 验收标准
1. 实现的离线 RL 算法在 D4RL 标准任务上有竞争力性能。
2. 能解释分布移位与外推误差的具体表现。
3. 理解约束优化如何缓解 OOD 问题。
